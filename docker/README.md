Spark 1.4.1, configured to accompany [Spark Succinctly](https://www.syncfusion.com/resources/techportal/details/ebooks/spark), a free Syncfusion eBook by [Marko Å valjek](https://twitter.com/msvaljek).

Usage, starting in the Spark REPL:

* docker run -it --rm --name spark sixeyed/spark-succinctly

Usage, starting in PySpark:

* docker run -it --rm --name spark sixeyed/spark-succinctly pyspark
